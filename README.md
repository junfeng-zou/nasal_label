# 鼻内镜手术视频标注系统

专为医学专业人士设计的鼻内镜手术视频标注工具，支持图像质量、内窥镜距器械距离、腔壁压迫风险和上下方向偏移评估。

## 功能特点

- 🎬 **视频播放**：内置视频播放器，支持多种格式
- 📷 **图像质量评估**：清晰/模糊/遮挡三级评估
- 📏 **内窥镜距器械距离评估**：过近/合适/过远三级评估
- ⚠️ **内窥镜压迫腔壁风险评估**：距左腔壁、距右腔壁两项评估
- ↕️ **上下方向偏移评估**：过于偏上/过于偏下/合适三级评估
- 💾 **自动保存**：标注数据自动保存，防止数据丢失
- 📊 **进度追踪**：实时显示标注进度
- 🖥️ **一键启动**：Windows环境下双击即用

## 标注维度

### 1. 图像质量（单选）
| 选项 | 说明 |
|------|------|
| 清晰 | 画面质量良好，细节可见 |
| 模糊 | 画面不清晰，影响判断 |
| 遮挡 | 视野被血液、组织等遮挡 |

### 2. 内窥镜距器械距离（单选）
| 选项 | 说明 |
|------|------|
| 过近 | 内窥镜距离器械过近 |
| 合适 | 距离适中 |
| 过远 | 内窥镜距离器械过远 |

### 3. 内窥镜压迫腔壁的风险评估（左右方向×三级）
分别评估距左腔壁、距右腔壁：
- **高风险**：内窥镜已贴近腔壁，疑似接触或压迫
- **中风险**：内窥镜明显靠近腔壁，但尚有一定安全距离
- **低风险**：内窥镜与腔壁距离合适，压迫风险较低

### 4. 上下方向偏移（单选）
| 选项 | 说明 |
|------|------|
| 过于偏上 | 视野或内窥镜位置过于偏上 |
| 过于偏下 | 视野或内窥镜位置过于偏下 |
| 合适 | 上下方向位置合适 |

## 目录结构

```
nasal_label/
├── annotation_app.py          # 标注主程序
├── config.json                # 标签配置文件
├── requirements.txt           # Python依赖
├── 启动标注程序.bat            # 启动脚本（医生使用）
├── 安装Python.bat              # Python环境安装（医生使用）
├── 医生使用说明.md             # 医生操作指南
├── merge_annotations.py       # 数据合并工具（研究员使用）
├── videos/                    # 视频文件夹
│   ├── video_001.mp4
│   ├── video_002.mp4
│   └── ...
└── annotations.json           # 标注数据（程序自动生成）
```

## 使用流程

### 分发给医生

1. 将分配给该医生的视频放入 `videos/` 文件夹
2. 打包所有文件发送给医生
3. 医生双击 `启动标注程序.bat` 即可使用

### 回收数据

1. 医生发回 `annotations.json`
2. 运行 `python merge_annotations.py` 合并多位医生的数据

## 自定义标签

编辑 `config.json` 文件可修改标签选项。
