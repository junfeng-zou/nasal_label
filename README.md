# 鼻内镜手术视频标注系统

专为医学专业人士设计的鼻内镜手术视频标注工具，支持图像质量、器械距离和腔壁风险评估。

## 功能特点

- 🎬 **视频播放**：内置视频播放器，支持多种格式
- 📷 **图像质量评估**：清晰/模糊/遮挡三级评估
- 📏 **器械距离评估**：五级距离评估（太远~太近）
- ⚠️ **腔壁风险评估**：四个方向（左/右/上/下）× 三级风险（危险/警告/安全）
- 💾 **自动保存**：标注数据自动保存，防止数据丢失
- 📊 **进度追踪**：实时显示标注进度
- 🖥️ **一键启动**：Windows环境下双击即用

## 标注维度

### 1. 图像质量（单选）
| 选项 | 说明 |
|------|------|
| 清晰 | 画面质量良好，细节可见 |
| 模糊 | 画面不清晰，影响判断 |
| 遮挡 | 视野被血液、组织等遮挡 |

### 2. 离器械距离（单选）
| 选项 | 说明 |
|------|------|
| 太远 | 器械在画面中过小 |
| 比较远 | 距离略远，尚可接受 |
| 合适 | 距离适中，细节清晰 |
| 比较近 | 距离略近，视野范围较小 |
| 太近 | 距离过近，无法看清整体 |

### 3. 腔壁风险评估（四方向×三级）
每个方向（左/右/上/下）独立评估：
- **危险**：存在明显损伤风险
- **警告**：存在潜在风险
- **安全**：处于安全状态

## 目录结构

```
nasal_label/
├── annotation_app.py          # 标注主程序
├── config.json                # 标签配置文件
├── requirements.txt           # Python依赖
├── 启动标注程序.bat            # 启动脚本（医生使用）
├── 安装Python.bat              # Python环境安装（医生使用）
├── 医生使用说明.md             # 医生操作指南
├── merge_annotations.py       # 数据合并工具（研究员使用）
├── videos/                    # 视频文件夹
│   ├── video_001.mp4
│   ├── video_002.mp4
│   └── ...
└── annotations.json           # 标注数据（程序自动生成）
```

## 使用流程

### 分发给医生

1. 将分配给该医生的视频放入 `videos/` 文件夹
2. 打包所有文件发送给医生
3. 医生双击 `启动标注程序.bat` 即可使用

### 回收数据

1. 医生发回 `annotations.json`
2. 运行 `python merge_annotations.py` 合并多位医生的数据

## 自定义标签

编辑 `config.json` 文件可修改标签选项。

## 技术栈
