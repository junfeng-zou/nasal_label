# 鼻内镜手术视频标注系统 - 医生使用指南

## 一、快速开始

### 第一步：安装Python环境（仅首次使用需要）
1. 双击运行 `安装Python.bat`
2. 在弹出的安装窗口中：
   - **重要**：勾选底部的 "Add Python to PATH" 选项
   - 点击 "Install Now" 进行安装
   - 等待安装完成

### 第二步：准备视频文件
1. 打开 `videos` 文件夹（如果没有请手动创建）
2. 将您需要标注的鼻内镜手术视频文件放入该文件夹
3. 支持的格式：MP4、AVI、MOV、MKV、WMV

### 第三步：启动标注程序
1. 双击运行 `启动标注程序.bat`
2. 程序会自动打开浏览器，进入标注界面
3. 如果浏览器没有自动打开，请手动访问：http://localhost:8501

## 二、标注内容说明

本系统需要标注以下四个维度的信息：

### 📷 1. 图像质量（单选）
评估当前视频画面的清晰程度：
- **清晰**：画面质量良好，细节可见
- **模糊**：画面不清晰，影响判断
- **遮挡**：视野被血液、组织等遮挡

### 📏 2. 内窥镜距器械距离（单选）
评估内窥镜与手术器械的距离：
- **过近**：内窥镜距离器械过近
- **合适**：距离适中，操作细节清晰可见
- **过远**：内窥镜距离器械过远

### ⚠️ 3. 内窥镜压迫腔壁的风险评估（左右方向，各单选）

对左右两个方向的腔壁距离进行风险评估：

| 方向 | 说明 |
|------|------|
| ⬅️ 距左腔壁 | 评估内窥镜距左侧腔壁的压迫风险 |
| ➡️ 距右腔壁 | 评估内窥镜距右侧腔壁的压迫风险 |

每个方向的风险等级：
- **即将接触**：内窥镜即将接触腔壁，需要立即注意
- **靠近**：内窥镜靠近腔壁，需要保持警惕
- **合适**：内窥镜距腔壁状态合适

### ↕️ 4. 上下方向偏移（单选）
评估画面或内窥镜在上下方向上的偏移情况：
- **过于偏上**：视野或内窥镜位置过于偏上
- **过于偏下**：视野或内窥镜位置过于偏下
- **合适**：上下方向位置合适

## 三、标注操作流程

### 1. 观看视频
- 在页面左侧可以看到视频播放器
- 使用播放器的控制按钮：播放、暂停、进度条

### 2. 进行标注
- 在页面右侧的标注区域，依次选择：
  - 图像质量状态
  - 内窥镜距器械距离
  - 距左腔壁、距右腔壁
  - 上下方向偏移

### 3. 保存标注
- 点击绿色的 **"💾 保存标注"** 按钮
- 看到绿色提示 "标注已保存" 即表示保存成功

### 4. 切换视频
- **方法一**：在左侧边栏的下拉菜单中选择视频
- **方法二**：使用底部的 "上一个"/"下一个" 按钮
- **方法三**：点击 "下个未标注" 快速定位未标注的视频

## 五、完成标注后

当您完成所有视频标注后：
1. 关闭浏览器窗口
2. 在弹出的命令行窗口中按任意键退出程序
3. 将 `annotations.json` 文件发送给研究团队

## 六、常见问题

### Q1：双击启动后弹出错误窗口？
**A**：请确认已运行 `安装Python.bat` 安装Python环境。

### Q2：浏览器没有自动打开？
**A**：请手动在浏览器地址栏输入：http://localhost:8501

### Q3：视频无法播放？
**A**：请确认视频格式为MP4/AVI/MOV/MKV/WMV中的一种。建议使用MP4格式。

### Q4：标注数据丢失了？
**A**：不会丢失。标注数据自动保存在 `annotations.json` 文件中。每次点击"保存标注"都会立即写入文件。

### Q5：可以修改已保存的标注吗？
**A**：可以。选择对应的视频，修改标注选项后再次点击"保存标注"即可覆盖原有数据。
