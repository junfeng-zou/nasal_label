# 分发清单

## 发送给每位医生的文件清单

### 必需文件
- [ ] `annotation_app.py` - 标注主程序
- [ ] `config.json` - 标签配置文件
- [ ] `requirements.txt` - Python依赖列表
- [ ] `启动标注程序.bat` - 启动脚本
- [ ] `安装Python.bat` - Python环境安装脚本
- [ ] `医生使用说明.md` - 操作指南
- [ ] `videos/` 文件夹 - 包含分配给该医生的视频文件

### 可选文件
- [ ] `test_app.py` - 测试脚本（可选）
- [ ] `README.md` - 项目说明（可选）

## 打包步骤

### 1. 准备视频文件
```bash
# 将500段视频按照分配计划分发给不同医生
# 示例：每位医生100段视频

# 医生A
mkdir -p package_doctorA/videos
cp video_001.mp4 ... video_100.mp4 package_doctorA/videos/

# 医生B
mkdir -p package_doctorB/videos
cp video_101.mp4 ... video_200.mp4 package_doctorB/videos/

# 以此类推...
```

### 2. 复制程序文件
```bash
# 为每位医生创建完整包
for doctor in doctorA doctorB doctorC; do
    cp annotation_app.py package_$doctor/
    cp config.json package_$doctor/
    cp requirements.txt package_$doctor/
    cp 启动标注程序.bat package_$doctor/
    cp 安装Python.bat package_$doctor/
    cp 医生使用说明.md package_$doctor/
done
```

### 3. 打包压缩
```bash
# 压缩为zip文件
zip -r 手术视频标注系统_医生A.zip package_doctorA/
zip -r 手术视频标注系统_医生B.zip package_doctorB/
```

## 数据回收流程

### 1. 接收标注文件
每位医生完成标注后，会发送 `annotations.json` 文件

### 2. 整理数据结构
``collected_annotations/
  ├── 医生A/
  │   └── annotations.json
  ├── 医生B/
  │   └── annotations.json
  └── 医生C/
      └── annotations.json
```

### 3. 合并数据
运行 `python merge_annotations.py` 生成汇总文件

## 注意事项

1. **视频格式**：建议统一使用MP4格式，确保兼容性
2. **文件命名**：视频文件名不要包含特殊字符和空格
3. **Python版本**：确保医生安装Python 3.8+
4. **数据备份**：建议在分发前备份原始视频文件
