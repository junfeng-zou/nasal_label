# Windows .exe 打包说明

## 打包流程（需要在 Windows 系统上执行）

### 方法一：使用 PyInstaller（推荐）

#### 1. 安装依赖
```powershell
pip install pyinstaller streamlit pandas
```

#### 2. 执行打包
```powershell
python build_exe.py
```

或直接运行 PyInstaller 命令：
```powershell
pyinstaller --name="手术视频标注系统" --onedir --clean launcher.py
```

#### 3. 输出文件
打包完成后，在 `dist/` 目录下生成：
- `手术视频标注系统/` 目录
- 包含 `手术视频标注系统.exe` 和所有依赖

---

### 方法二：使用 NSIS 安装程序（更专业）

#### 1. 安装 pynsist
```powershell
pip install pynsist
```

#### 2. 运行打包
```powershell
pynsist installer.cfg
```

#### 3. 输出文件
生成 `手术视频标注系统_安装程序.exe`，医生运行此安装程序后即可使用。

---

## 分发给医生

### 打包后的文件结构
```
手术视频标注系统/
├── 手术视频标注系统.exe    # 主程序（双击运行）
├── annotation_app.py       # 标注应用
├── config.json             # 配置文件
├── videos/                 # 视频目录（放入要标注的视频）
├── _internal/              # Python运行环境（自动生成）
│   ├── python311.dll
│   ├── ...
└── annotations.json        # 标注数据（运行后生成）
```

### 分发步骤
1. 将 `dist/手术视频标注系统/` 整个目录压缩为 zip
2. 发送给医生
3. 医生解压后，将视频放入 `videos/` 目录
4. 双击 `手术视频标注系统.exe` 即可运行

---

## 注意事项

1. **打包必须在 Windows 上进行**
   - PyInstaller 不支持跨平台打包
   - 在 Linux/Mac 上打包的 .exe 无法在 Windows 上运行

2. **打包体积**
   - 单文件模式（--onefile）：约 150-200 MB
   - 单目录模式（--onedir）：约 200-250 MB，但启动更快

3. **首次启动较慢**
   - 单文件模式首次启动需要解压，约 10-20 秒
   - 单目录模式启动较快，约 3-5 秒

4. **视频文件**
   - 打包时不会包含 videos 目录中的视频
   - 需要在分发时单独提供或让医生自行放置
