# Windows .exe 打包说明

## 打包流程（需要在 Windows 系统上执行）

### 方法一：使用 PyInstaller（推荐）

#### 1. 安装依赖
```powershell
python -m pip install --upgrade pip
pip install -r requirements.txt pyinstaller
```

#### 2. 执行打包
```powershell
python build.py
```

或直接运行 PyInstaller 命令：
```powershell
pyinstaller build.spec --clean --noconfirm
```

#### 3. 输出文件
打包完成后，在 `dist/` 目录下生成：
- `手术视频标注系统/` 目录
- 包含 `手术视频标注系统.exe` 和所有依赖

---

### 方法二：使用 NSIS 安装程序（更专业）

#### 1. 安装 pynsist
```powershell
pip install pynsist
```

#### 2. 运行打包
```powershell
pynsist installer.cfg
```

#### 3. 输出文件
生成 `手术视频标注系统_安装程序.exe`，医生运行此安装程序后即可使用。

---

## 分发给医生

### 打包后的文件结构
```
手术视频标注系统/
├── 手术视频标注系统.exe    # 主程序（双击运行）
├── config.json             # 可编辑配置文件
├── videos/                 # 视频目录（放入要标注的视频）
├── 使用说明.txt
├── _internal/              # Python运行环境（自动生成）
│   ├── annotation_app.py
│   ├── python311.dll
│   ├── ...
└── annotations.json        # 标注数据（运行后生成）
```

### 分发步骤
1. 将 `dist/手术视频标注系统/` 整个目录压缩为 zip
2. 发送给医生
3. 医生解压后，将视频放入 `videos/` 目录
4. 双击 `手术视频标注系统.exe` 即可运行

`使用说明.txt` 来自项目根目录同名文件。需要修改医生端简要说明时，直接编辑根目录的 `使用说明.txt` 后重新打包。

---

## 注意事项

1. **打包必须在 Windows 上进行**
   - PyInstaller 不支持跨平台打包
   - 在 Linux/Mac 上打包的 .exe 无法在 Windows 上运行

2. **打包体积**
   - 单文件模式（--onefile）：约 150-200 MB
   - 单目录模式（--onedir）：约 200-250 MB，但启动更快

3. **首次启动较慢**
   - 单文件模式首次启动需要解压，约 10-20 秒
   - 单目录模式启动较快，约 3-5 秒

4. **视频文件**
   - 打包时不会包含 videos 目录中的视频
   - 需要在分发时单独提供或让医生自行放置

## macOS 打包

macOS 版本需要在 macOS 环境中打包，不能在 Windows 上直接生成。项目已提供 GitHub Actions workflow：

```
.github/workflows/build-macos.yml
```

### 产物

workflow 会分别生成：

- `nasal-label-macos-arm64.zip`：适用于 Apple Silicon Mac
- `nasal-label-macos-intel.zip`：适用于 Intel Mac

压缩包结构：

```
手术视频标注系统-macOS/
├── 手术视频标注系统.app
├── videos/
├── config.json
├── 使用说明.txt
├── macOS打开说明.txt
└── annotations.json        # 标注数据（运行后生成）
```

### 使用方式

1. 解压对应架构的 zip
2. 将视频放入 `videos/` 目录
3. 双击 `手术视频标注系统.app`

如果 macOS 提示无法验证开发者，先右键点击 app，选择“打开”。如果仍无法打开，查看压缩包内的 `macOS打开说明.txt`。
